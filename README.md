Features

Add & delete tasks with smooth slide animations
Priority levels — High, Medium, Low with color-coded tags
Check off tasks with an animated checkbox
Reorder tasks up and down
Filter tasks by All, Active, or Completed
Progress bar that updates as you complete tasks
Stats counter showing total, done, and remaining tasks
Clear completed button to clean up finished tasks
Keyboard support — press Enter to add a task
Fully responsive layout


Tech Stack

React 18
JavaScript (ES6+)
CSS3 — animations, transitions, flexbox, grid
Vite


Getting Started
Prerequisites

Node.js installed on your machine
npm or yarn

Installation

Clone the repo

bashgit clone https://github.com/citygirlsf/todo-app.git

Go into the project folder

bashcd todo-app

Install dependencies

bashnpm install

Start the development server

bashnpm run dev

Open your browser and go to http://localhost:5173
